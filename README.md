🎮 TicTacToe Pro

Современная браузерная игра «Крестики-нолики», разработанная с использованием JavaScript и Vite. Проект создан в рамках учебной практики и демонстрирует применение модульной архитектуры, работу с DOM, хранение данных в браузере и реализацию игрового искусственного интеллекта.

🌐 Ссылки

Netlify: https://thunderous-valkyrie-9421d6.netlify.app/

✨ Возможности

Доступно два режима игры: Игрок vs Компьютер и Игрок vs Игрок.

Компьютерный соперник поддерживает три уровня сложности: легкий, средний и сложный.

Игра автоматически определяет победителя или ничью, подсвечивает выигрышную комбинацию и отображает результат в модальном окне.

Статистика побед и ничьих сохраняется в LocalStorage, поэтому результаты не теряются после обновления страницы.

Приложение поддерживает светлую и темную темы оформления, а также корректно отображается как на компьютерах, так и на мобильных устройствах.

💡 Особенности проекта

Во время разработки основной задачей было создать не просто классические крестики-нолики, а современное веб-приложение с удобным интерфейсом и чистой структурой кода.

Игровая логика полностью отделена от пользовательского интерфейса, благодаря чему проект легко поддерживать и расширять.

Компьютерный противник реализован с несколькими уровнями сложности. На максимальном уровне сложности он анализирует игровое поле, блокирует потенциальные победные комбинации игрока и выбирает наиболее выгодные ходы.

🚀 Установка и запуск

Клонировать репозиторий

git clone ВСТАВЬ_ССЫЛКУ_НА_GITHUB

Перейти в папку проекта

cd tictactoe-pro

Установить зависимости

npm install

Запустить проект

npm run dev

Создать production-сборку

npm run build

Предпросмотр production-версии

npm run preview
